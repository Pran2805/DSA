# Exception Handling
## Java Exceptions
An exception is an unexpected event that occurs during program execution.
It affects the flow of the program instruction which can cause the program to terminate abnormally.

### An exception can occur for many reason. Some of them are:
    - Invalid User Input
    - Device Failure
    - Loss of Network Connnection
    - Physical Limitation (out of disk memory)
    - Code errors
    - Opening an unavailable file

## Java Runtime Exception
A runtime exception happen due to a programming error.
They are also known as unchecked exception.
#### Eg.
    - Null Pointer Exception
    - ArrayIndexOutOfBoundsException
    - Arithmetic Acception

## Java IO Exception Exception
An IO exception is also known as a check exception. 
They are checked by the compiler at the compile time and the programmer is prompted to handle these exception.
